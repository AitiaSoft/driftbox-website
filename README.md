# DriftBox — Never Let a Conversation Drift Away

Product website for DriftBox, an AI-powered communication intelligence platform by AitiaSoft.

## Overview

DriftBox captures conversations across email, Slack, Teams, and WhatsApp, then surfaces what matters using AI — summaries, action items, smart alerts, and natural language search.

## Tagline

**"Never let a conversation drift away."**

## Logo Concept

The logo features two overlapping speech bubbles on a purple/indigo gradient background, representing:
- Multiple communication channels converging into one
- Conversations being captured (not drifting away)
- The "box" that holds everything together

## Site Structure

- `index.html` — Single-page product website
- `styles.css` — Dark theme, modern startup design
- `script.js` — Animations, scroll effects, waitlist form
- `favicon.svg` — SVG favicon with logo

## Design

- Dark theme (#0A0A0F background)
- Indigo/purple gradient accent (#6366F1 → #8B5CF6)
- Inter font family
- Responsive (mobile-first)
- Scroll animations
- Interactive app preview mockup
